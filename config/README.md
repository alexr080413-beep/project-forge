# Config

Store safe, version-controlled configuration templates and examples here.

Do not commit real credentials, tokens, private keys, personal paths, or environment-specific secrets.

`profiles.example.yaml` contains safe example Profile Manager definitions for MWTC, ITX, and Joint Exercise environments.

`integration_sources.example.yaml` contains safe example Integration Service source definitions for RSS, website, manual upload, local file, and placeholder email, social media, SharePoint, and API sources. These examples are for validation and dry-run collection only.
