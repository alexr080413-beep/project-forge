# Logo

Forge uses a metallic shield logo with an integrated anvil base and abstract forged `F` mark.

Primary logo asset:

```text
assets/forge-logo.png
```

Legacy placeholder asset:

```text
assets/project-forge-logo-legacy-placeholder.svg
```

The legacy placeholder must not be used as the primary README, presentation, website, or Forge Studio identity mark.

## Logo Concept

| Element | Meaning |
| --- | --- |
| Shield | Control, protection, security, and mission trust. |
| Anvil | Forge as a production system that shapes raw information into controlled exercise products. |
| Abstract F | Product recognition without relying on text inside the mark. |
| Metallic surface | Enterprise durability, seriousness, and command-center precision. |
| Orange glow | Forge energy, operational urgency, and active workflow state. |

## Usage

- Use the logo on dark backgrounds when possible.
- Pair the logo with the slogan "Every Event. Every Inject. Every Exercise." on public-facing materials.
- Pair the logo with the tagline "Human Command. Machine Assistance." when explaining the human-machine relationship.
- Keep sufficient empty space around the mark so the shield remains legible.
- Use the mark at sizes where the shield, anvil base, and F geometry remain readable.

## Clear Space

Use clear space equal to at least 15 percent of the logo width on all sides. Do not place text, badges, diagrams, or UI controls inside that area.

## Minimum Size

| Context | Minimum Width |
| --- | --- |
| README hero | 180px |
| Presentation title slide | 120px |
| Website header | 40px |
| App shell mark | 28px |
| Favicon or tiny icon | Use a simplified future icon instead of the detailed raster mark. |

## Do

- Use the primary PNG for GitHub README and presentation title pages.
- Use dark graphite or black surfaces behind the mark.
- Use orange accent light sparingly around the logo.
- Preserve the original aspect ratio.

## Do Not

- Do not stretch, skew, crop, or rotate the logo.
- Do not recolor the logo into unrelated palettes.
- Do not place it on busy imagery.
- Do not add flags, weapons, unit insignia, or additional symbols to the mark.
- Do not use it as a faint watermark behind operational UI content.
- Do not add slogan or tagline text inside the image.
