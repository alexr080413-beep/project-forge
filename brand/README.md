# Forge Brand Kit

This directory is the official brand system for Forge. It defines the identity, messaging, visual language, UI direction, imagery guidance, and reusable marketing copy for future flyers, pitch decks, GitHub Pages, Forge Studio surfaces, and public project materials.

Forge is an open-source digital operations platform for Exercise Control.

## Brand Foundation

| Element | Standard |
| --- | --- |
| Product name | Forge |
| Primary positioning | Open-Source Digital Operations Platform for Exercise Control |
| Aspirational positioning | The Operating System for Modern Exercise Control |
| Primary slogan | Every Event. Every Inject. Every Exercise. |
| Secondary tagline | Human Command. Machine Assistance. |
| Core philosophy | People create training. Forge manages information. |
| Mission | Forge modernizes Exercise Control by providing an open, modular platform that connects planning, execution, intelligence, assessment, and after-action review into one operational workflow. |
| Vision | Become the digital operating system for military exercise control. |

## Brand Principles

- Human judgment is authoritative.
- Forge assists; people decide.
- Open architecture.
- Mission before technology.
- Modular by design.
- Secure by default.
- Built for military, joint, and coalition exercises.

## Usage Rules

- Use "Forge" as the product name.
- Use the primary positioning for public, GitHub, and introductory materials.
- Use the aspirational positioning for executive summaries and vision sections.
- Pair the slogan with the logo when space allows.
- Use the secondary tagline when explaining the human-machine relationship.
- Avoid language that implies Forge replaces controllers, automates release authority, or publishes products without human review.
- Keep the visual tone dark, metallic, operational, and disciplined.

## Source Alignment

This kit aligns with the repository README, Forge Studio UX blueprint, Forge Studio design system, Exercise Lifecycle model, and platform architecture documentation. When the product architecture changes, update this brand kit so external messaging stays consistent with the actual Forge system.
