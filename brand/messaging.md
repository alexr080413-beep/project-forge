# Messaging

This file defines approved messaging for Forge audiences, channels, and reusable marketing materials.

## Core Message

Forge is an open-source digital operations platform for Exercise Control. It helps teams connect planning, execution, intelligence, product workflows, human review, assessment, and after-action records into one operational workflow.

## Message Pillars

| Pillar | Message |
| --- | --- |
| Control | Human controllers retain authority over exercise products, approvals, corrections, and release decisions. |
| Tempo | Forge reduces friction between signal intake, scenario context, translation, drafting, QA, and review. |
| Traceability | Products and decisions preserve source references, context, QA findings, review history, audit records, and metrics. |
| Modularity | Forge is built around services, profiles, plugins, and workflows that can evolve without collapsing into one opaque system. |
| Exercise lifecycle | Forge organizes work across planning, preparation, execution, assessment, and archive. |

## Audience Guidance

| Audience | Emphasize | Avoid |
| --- | --- | --- |
| Exercise Directors | Command visibility, release authority, review pressure, exercise timeline, after-action traceability. | Claims that Forge runs the exercise or replaces command judgment. |
| Intelligence Chiefs | Source traceability, entity lookup, scenario translation, intelligence products, review confidence. | Claims that Forge creates real intelligence or bypasses analytic review. |
| Exercise Control Officers | Controller workspaces, inject library, common operational picture, review queues, workflow status. | Generic productivity or dashboard language. |
| Training Command Leaders | Repeatable exercise workflows, archive, assessment, readiness value, open architecture. | Vendor-style promises, vague AI transformation language, or unsupported ROI claims. |
| Joint Exercise Planners | Profiles, coalition-aware mappings, modular plugins, controlled workflows, shared operating picture. | Service-specific assumptions that exclude joint or coalition use. |
| Senior Defense Leaders | Modernization, governance, audit trail, scalability, human authority, lifecycle integration. | Technical deep dives before the mission value is clear. |

## Example Copy

### Executive One-Pager

Forge is an open-source digital operations platform for Exercise Control. It connects planning, execution, intelligence, assessment, and after-action review into one operational workflow so controllers can manage the full exercise lifecycle with stronger traceability and clearer authority.

Forge is designed for military, joint, and coalition exercises where information moves quickly and training products must remain scenario-consistent. The platform organizes controller workspaces, operational timelines, inject libraries, human review queues, audit trails, and exercise archives around one common exercise record.

Human judgment remains authoritative. Forge assists; people decide.

### Technical One-Pager

Forge is a modular platform foundation for Exercise Control workflows. It separates source intake, storage, scenario context, entity lookup, event creation, decisioning, translation, AI-assisted reasoning stubs, product formatting, QA, review, distribution dry-runs, audit logging, metrics, configuration, automation, and security into explicit service boundaries.

The current foundation is local and deterministic. It performs no external API calls, no real OpenAI calls, no web scraping, no email delivery, and no automatic release into exercise play. Future capabilities can extend the platform through profiles, product plugins, workflow modules, and controlled integrations.
