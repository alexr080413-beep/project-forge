# Voice

Forge should sound confident, precise, operational, and mission-focused.

## Voice Attributes

| Attribute | Meaning |
| --- | --- |
| Confident | Speak with clarity. Do not oversell or hedge the core mission. |
| Precise | Use exact operational terms and avoid vague claims. |
| Operational | Write for controllers, planners, reviewers, and leaders doing real work. |
| Mission-focused | Lead with exercise outcomes before technology. |
| Controlled | Reinforce human authority, auditability, and safe boundaries. |

## Writing Rules

- Use "Forge" as the product name.
- Use "Exercise Control" for the function and "controllers" for the people operating the workflow.
- Prefer "assist", "prepare", "organize", "validate", "route", "record", and "surface".
- Avoid saying Forge "commands", "decides", "publishes", or "releases" without a human controller.
- Keep sentences short and direct.
- Use concrete nouns: exercise lifecycle, operational timeline, controller workspace, inject library, review queue, audit trail, exercise archive.
- Explain AI assistance as bounded support inside human-controlled workflows.

## Preferred Language

Use:

- Exercise lifecycle
- Operational timeline
- Controller workspace
- Common operational picture
- Inject library
- Human review queue
- Complete audit trail
- Exercise archive
- Scenario-consistent product
- Source traceability
- Review authority
- After-action review

Avoid:

- Buzzwords
- Startup language
- Generic SaaS language
- Overhype
- AI-first positioning
- Autonomy claims
- Consumer app language
- Decorative metaphors

## Tone Examples

Too vague:

```text
Forge revolutionizes exercise workflows with next-generation AI productivity.
```

Use:

```text
Forge helps controllers move from signal to scenario-consistent product with source traceability, QA, human review, and audit records.
```

Too autonomous:

```text
Forge decides which products should be released into the exercise.
```

Use:

```text
Forge prepares and routes products for human review before release into exercise play.
```

Too generic:

```text
Forge is a powerful dashboard for teams.
```

Use:

```text
Forge provides controller workspaces, an operational timeline, review queues, metrics, and audit records for Exercise Control.
```

## Voice By Context

| Context | Voice Guidance |
| --- | --- |
| GitHub README | Clear, direct, open-source, architecture-aware. |
| Executive brief | Mission value first, concise, authority-focused. |
| Technical brief | Modular services, deterministic behavior, boundaries, testability. |
| Forge Studio UI | Compact labels, operational verbs, no marketing copy inside tools. |
| Presentations | Strong headlines, short supporting copy, concrete workflow examples. |
