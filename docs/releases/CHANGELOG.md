# Forge Release History

This changelog follows the spirit of [Keep a Changelog](https://keepachangelog.com/) and semantic versioning.

## [Unreleased]

### Added

- Governance foundation with ADR process, RFC process, engineering standards, definition of done, GitHub templates, support policy, and example exercise documentation.

## [v0.1.0] - Foundation Release

### Added

- Core Forge service foundations.
- Forge Studio MVP domain model and local web application.
- Exercise Data Engine.
- Exercise Workspace Framework.
- Project Atlas Exercise Designer.
- Exercise Relationship Engine.
- Operational Knowledge Graph.
- Forge Academy documentation system.
- Brand kit, design system, roadmap, and vision documentation.

### Notes

This release represents the Forge foundation baseline. It is local, deterministic, documentation-rich, and intentionally avoids production authentication, external integrations, real publishing, and live AI calls.
