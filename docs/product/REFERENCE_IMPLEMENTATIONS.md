# Reference Implementations

Reference implementations keep Forge development grounded in realistic, repeatable exercise workflows.

## Official Reference Exercise

Project Sentinel is the official Forge reference exercise.

Project Sentinel uses:

- Organization: Marine Corps Mountain Warfare Training Center.
- Exercise: Mountain Exercise 3-27.
- Status: Planning.
- Exercise Director: Colonel Smith.
- Exercise Control: Bridgeport EXCON.
- Training Audience: Infantry Battalion.

See [examples/sentinel/README.md](../../examples/sentinel/README.md).

## Required Validation

Every feature, UI change, tutorial, regression test, screenshot, Academy lesson, and demonstration must be validated against Project Sentinel when applicable.

## What Sentinel Validates

Sentinel validates:

- Exercise setup.
- Atlas planning.
- Objectives.
- Controllers.
- 50-inject library.
- Timeline.
- Intelligence and ISR.
- Weather.
- Cyber activity.
- Logistics and medical support.
- Media and public affairs.
- Products.
- Observations.
- AAR findings.
- Knowledge Graph relationships.

## Extension Rule

If a future feature requires new reference data, extend Project Sentinel first unless the feature requires a different exercise family.

If a separate reference exercise is needed, document why Sentinel cannot cover the case.
