# Publish To Mission Control

Use this guide to understand the planned publish workflow from Atlas to Mission Control.

## Steps

1. Open `Exercise Designer`.
2. Validate objectives, controllers, timeline, review requirements, and publish readiness.
3. Confirm the Exercise Director approves the plan.
4. Select `Publish to Mission Control`.
5. Confirm live workspaces receive the approved exercise objects.

## Current Boundary

Real publishing is not implemented. The toolbar control is a placeholder. Publishing must remain explicit and human-approved.
