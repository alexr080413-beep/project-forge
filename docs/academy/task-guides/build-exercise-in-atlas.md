# Build Exercise In Atlas

Use Atlas to draft the exercise structure before execution.

## Steps

1. Open `Exercise Designer`.
2. Review the Object Library.
3. Add or inspect objectives.
4. Add controllers.
5. Add injects and planned timeline events.
6. Select a planned item.
7. Review linked objectives, related injects, assigned controller, produced products, follow-on events, and validation warnings.

## Current Boundary

Atlas currently displays mock planning objects and relationship data. Persistence and drag and drop are not implemented.
