# Validate Exercise

Use this guide to check exercise readiness.

## Steps

1. Open `Exercise Designer`.
2. Review validation status.
3. Check objectives linked.
4. Check controllers assigned.
5. Review timeline conflicts.
6. Confirm review requirements.
7. Confirm publish readiness.

## Current Boundary

Validation status is mock data in the current Atlas framework. Future versions should calculate validation from the persisted exercise graph.
