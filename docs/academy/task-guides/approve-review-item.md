# Approve Review Item

Use this guide to approve an item in the Review Queue.

## Steps

1. Open `Review Queue`.
2. Locate the pending item.
3. Review title, item type, status, reviewer, and comments.
4. Select `Approve`.
5. Confirm status updates in Review Queue, Inject Library or Exercise Library, Dashboard, and Audit.

## Human Control Point

Approval is a release authority action. Do not approve items without reviewing scenario fit and traceability.
