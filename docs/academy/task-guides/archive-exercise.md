# Archive Exercise

Use this guide to archive an exercise after execution and assessment.

## Steps

1. Confirm execution is complete.
2. Review pending approvals.
3. Confirm products are captured in the Exercise Library.
4. Review Knowledge Graph traceability for objectives, observations, and AAR findings.
5. Select `Archive`.
6. Confirm the exercise status changes to archived.

## Archive Standard

Archive only after the exercise record is complete enough to support AAR, search, and future reuse.
