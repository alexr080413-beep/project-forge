# Mission Control 101

Mission Control is the operational picture for the selected exercise.

## What You See

- Active exercise.
- Exercise status and phase.
- Live execution state.
- Current exercise time.
- Execution controls.
- Controller status.
- Pending reviews.
- Open injects.
- Active injects.
- Upcoming timeline events.
- Execution alerts.
- Products generated.
- Timeline summary.
- Latest activity.

## What It Is For

Use Mission Control to answer: "What is happening in the exercise right now, and what needs human attention?"

During live execution, Mission Control is also where authorized users start, pause, resume, end, and archive the exercise. These controls update the shared Exercise Data Engine, which then updates Timeline, Inject Library, Controllers, Review Queue, Analytics, Activity Feed, and Audit.

## Human Control Point

Mission Control may show pending work, but it does not bypass review authority. Inject release, review approval, and exercise state changes remain deliberate human actions.
