# Exercise Library 101

The Exercise Library is the repository for products generated during an exercise.

## What You See

- Product type.
- Title.
- Status.
- Version.
- Last updated.
- Author.
- Review status.

## What It Is For

Use the Exercise Library to inspect products, metadata, version history, review state, and archive status.

## AAR Connection

Products in the Exercise Library help explain what was produced, when, by whom, and under what review status.
