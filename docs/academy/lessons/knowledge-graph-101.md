# Knowledge Graph 101

The Knowledge Graph is the connected intelligence layer of Forge.

## What You See

- Operational assets as graph nodes.
- Relationships as graph edges.
- Asset type filters.
- Search.
- Node Inspector.
- Relationship highlighting.

## What It Is For

Use Knowledge Graph to understand how objectives, injects, products, controllers, timeline events, observations, and AAR findings relate.

## Current Boundary

The current graph uses realistic mock data. It does not implement AI, Mission Replay, persistent graph storage, or analytics.
