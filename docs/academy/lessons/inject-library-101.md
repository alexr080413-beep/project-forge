# Inject Library 101

The Inject Library stores exercise injects and their review state.

## What You See

- Inject title.
- Category.
- Status.
- Scheduled time.
- Controller.
- Priority.

## What It Is For

Use the Inject Library to create, edit, assign, schedule, approve, reject, and track injects.

## Human Control Point

No inject should be treated as releasable without explicit human approval.
