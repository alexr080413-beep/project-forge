# Review Queue 101

The Review Queue is the human approval gate for Forge.

## What You See

- Pending items.
- Approved items.
- Rejected items.
- Reviewer.
- Timestamp.
- Decision and comments.

## What It Is For

Use the Review Queue to approve, reject, or return items for revision.

## Human Control Point

Review decisions are authoritative records and should be audit-ready.
