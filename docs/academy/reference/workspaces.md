# Workspaces

Forge Studio workspaces are views into the selected Exercise.

| Workspace | Purpose |
| --- | --- |
| Mission Control | Real-time operational picture and exercise health. |
| Exercise Designer | Project Atlas planning and validation environment. |
| Knowledge Graph | Operational asset graph and relationship explorer. |
| Timeline | Chronological exercise events and controller updates. |
| Intelligence | Intelligence picture and product recommendations. |
| Inject Library | Inject creation, assignment, scheduling, and status. |
| Exercise Library | Product repository and archive. |
| Controllers | Controller assignments, workload, and status. |
| Review Queue | Human approval, rejection, and revision workflow. |
| Reports | Reporting framework for controller and AAR products. |
| Analytics | Metrics derived from exercise data. |
| Administration | Organization, settings, plugins, users, and audit administration. |
