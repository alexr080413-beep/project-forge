# Workspaces

Forge Studio workspaces are views into the selected Exercise.

| Workspace | Purpose |
| --- | --- |
| Mission Control | Live operational picture, exercise health, execution controls, alerts, upcoming events, activity, and current exercise time. |
| Exercise Designer | Project Atlas interactive planning environment for exercises, objectives, controllers, injects, timeline events, validation, graph relationships, package publication, and version history. |
| Knowledge Graph | Operational asset graph and relationship explorer. |
| Timeline | Chronological exercise events, execution states, event actions, and controller notes. |
| Intelligence | Intelligence picture and product recommendations. |
| Inject Library | Inject creation, assignment, scheduling, review status, release, acknowledgement, completion, and revision workflow. |
| Exercise Library | Product repository and archive. |
| Controllers | Controller assignments, assigned injects, upcoming events, task status, workload, and notes. |
| Review Queue | Human approval, rejection, revision, and approve-and-release workflow. |
| Reports | Reporting framework for controller and AAR products. |
| Analytics | Metrics derived from exercise data, including execution tempo, released injects, completed events, delayed events, pending reviews, and controller workload. |
| Administration | Organization, settings, plugins, users, and audit administration. |
