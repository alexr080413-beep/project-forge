# Glossary

| Term | Definition |
| --- | --- |
| Academy | Forge training, walkthrough, task-guide, reference, and troubleshooting system. |
| Atlas | Exercise Designer planning environment. |
| Exercise | Top-level operational container for a controlled training event. |
| Exercise Package | Versioned package produced by the Atlas Publish Pipeline containing the validated exercise, objectives, timeline, injects, controllers, operational assets, Knowledge Graph, relationships, validation summary, publication timestamp, and version number. |
| Inject | Planned stimulus introduced by Exercise Control. |
| Mission Control | Operational dashboard for the active exercise. |
| Operational Asset | Exercise object represented as a Knowledge Graph node. |
| Product | Generated or archived exercise artifact. |
| Review Queue | Human approval gate for injects and products. |
| Knowledge Graph | Connected graph of operational assets and relationships. |
| AAR | After-action review. |
