# Operational Assets

Operational Assets are graph nodes in the Forge Operational Knowledge Graph.

| Asset | Description |
| --- | --- |
| Exercise | Root context for the graph. |
| Objective | Training outcome and assessment anchor. |
| Inject | Planned or executed stimulus. |
| Timeline Event | Scheduled or observed exercise event. |
| Controller | Human owner or review authority. |
| Organization | Owning command, venue, lab, or joint environment. |
| Unit | Training audience or participating unit. |
| Product | Generated exercise artifact. |
| Intelligence Update | Scenario-consistent intelligence input. |
| Weather Event | Environmental condition. |
| Media Event | Simulated public affairs or media activity. |
| Decision Point | Commander or controller decision marker. |
| Observation | Captured evidence. |
| AAR Finding | After-action finding tied to evidence. |
| Template | Reusable planning pattern. |

## Atlas Alpha Editable Assets

Atlas Alpha supports local editing for:

- Exercise properties.
- Objectives with priority, success criteria, and linked assets.
- Controller assignments with role, responsibilities, linked objectives, and linked injects.
- Inject plans with controller, objective, priority, notes, and planned time.
- Timeline events with category and scheduled time.

These edits update the Exercise Data Engine and the Knowledge Graph for the running session.
