# Review Queue Troubleshooting

## Symptoms

- Review count does not change.
- Approved item still appears pending.
- Product review state does not match the queue.

## Checks

1. Confirm the item was approved, rejected, or returned for revision.
2. Confirm the selected exercise did not change.
3. Check Dashboard, Review Queue, Inject Library, Exercise Library, and Audit.
4. Restart the local app if mock state became confusing.

## Human Control Point

Review actions should be explicit and auditable.
