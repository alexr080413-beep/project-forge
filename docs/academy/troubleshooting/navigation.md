# Navigation Troubleshooting

## Symptoms

- A workspace appears empty.
- Exercise data does not match expectations.
- Breadcrumbs show the wrong workspace.

## Checks

1. Confirm the selected organization.
2. Confirm the selected exercise.
3. Use the sidebar to re-open the workspace.
4. Refresh the page if local state looks stale.

## Key Concept

Forge Studio follows `Forge -> Organization -> Exercise -> Workspace`.
