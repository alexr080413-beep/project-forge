# Definition Of Done

Every Forge feature is complete only when all relevant items are complete.

## Required Checklist

- [ ] Implementation complete.
- [ ] Tests pass.
- [ ] Documentation updated.
- [ ] Forge Academy updated.
- [ ] Task Guides updated.
- [ ] Reference updated.
- [ ] Troubleshooting updated.
- [ ] ADR updated if architecture changed.
- [ ] RFC linked if this is a major feature.
- [ ] Changelog updated.
- [ ] Screenshots updated if UI changed.

## Notes

Not every item applies to every change. If an item does not apply, the pull request should say why.

Documentation-only changes should still run diff hygiene and any relevant documentation checks.
