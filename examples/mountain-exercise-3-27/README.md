# Mountain Exercise 3-27

## Scenario

A mountain warfare training exercise at Bridgeport EXCON focused on command and control, route disruption, weather effects, and decision-making under constrained terrain.

## Training Audience

2d Battalion, 8th Marines.

## Objectives

- Exercise command and control in complex mountain terrain.
- Validate route disruption and casualty evacuation procedures.
- Assess intelligence, media, cyber, and weather inject response.

## Expected Products

- Intelligence baseline.
- Weather impact advisory.
- Civilian protest inject card.
- Commander decision record.
- AAR finding summary.

## Example Injects

- Civilian protest.
- Weather impact inject.
- GPS interference.
- Media interview.
- Avalanche warning.
