# Information Warfare

## Scenario

A notional exercise focused on information environment monitoring, media injects, cyber disruption, and decision-making under ambiguous public narratives.

## Training Audience

Information operations cell, intelligence staff, cyber cell, public affairs, and Exercise Control.

## Objectives

- Detect and assess information environment shifts.
- Coordinate cyber, intelligence, and media responses.
- Preserve scenario fidelity while producing controlled exercise products.

## Expected Products

- Information environment summary.
- Cyber incident update.
- Media response draft.
- Intelligence assessment.
- AAR finding on coordination.

## Example Injects

- Simulated viral post.
- Spoofed GPS report.
- Conflicting media interview.
- Cyber service degradation.
- Partner-nation information request.
