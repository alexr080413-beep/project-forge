<h2>What Project Forge Does</h2>

Project Forge uses real-world reporting as inspiration, then rewrites it through the lens of an exercise knowledge base: Road to War, base documents, fictional countries, ORBATs, training objectives, and approved report templates.

> Project Forge does not create real-world intelligence products. It creates notional exercise reports and injects for training audiences.
